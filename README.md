# Fedora-server

## Quick start

```
rm -rf Fedora-server
git clone https://github.com/trifoil/Fedora-server.git
cd Fedora-server
sudo sh main.sh
cd ..
clear
```

## Available in the bundle :

Installed from the script and ```required``` for the following services : 

* Docker

Admin services :

* Portainer
* Nginx Proxy Manager
* DDNS updater
* Uptime-kuma
* FileBrowser

User services :

* Nextcloud AIO
* Streaming
    * Jellyfin
    * Deluge
* Gitbucket
